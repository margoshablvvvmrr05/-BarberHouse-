# BarberHouse — Сайт барбершопу

Сайт розроблено студенткою **Маргаритою Булавіною**, група **ФТБ 4-9**

## Структура проєкту

```
barberhouse/
├── index.html        ← головний HTML-файл
├── css/
│   └── style.css     ← всі стилі сайту
├── js/
│   └── main.js       ← вся логіка та дані
└── README.md         ← цей файл
```

## Функціонал сайту

- **Головна** — hero-секція, статистика, про нас, галерея, відгуки
- **Послуги** — повний прайс-лист з фільтрами та калькулятором вартості
- **Галерея** — портфоліо робіт з фільтрацією по категоріях
- **Відгуки** — всі відгуки клієнтів з рейтингом
- **Контакти** — контактна інформація + форма онлайн-запису

## Як опублікувати на GitHub Pages

### Крок 1 — Створіть репозиторій
1. Зайдіть на [github.com](https://github.com) та увійдіть в акаунт
2. Натисніть **"New repository"** (зелена кнопка)
3. Введіть назву, наприклад: `barberhouse`
4. Поставте галочку **"Public"**
5. Натисніть **"Create repository"**

### Крок 2 — Завантажте файли
**Варіант А — через браузер (простіше):**
1. На сторінці репозиторію натисніть **"uploading an existing file"**
2. Перетягніть усі файли та папки (`index.html`, `css/`, `js/`)
3. Натисніть **"Commit changes"**

**Варіант Б — через Git (термінал):**
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГІН/barberhouse.git
git push -u origin main
```

### Крок 3 — Увімкніть GitHub Pages
1. Відкрийте репозиторій на GitHub
2. Перейдіть у **Settings** → **Pages**
3. У розділі **"Branch"** оберіть `main` та `/ (root)`
4. Натисніть **"Save"**
5. Через 1-2 хвилини сайт буде доступний за адресою:  
   `https://ВАШ_ЛОГІН.github.io/barberhouse/`

## Локальний запуск

Просто відкрийте `index.html` у браузері — сайт повністю працює без сервера.
